# Master User Journey Map

## Persona Paths

### 1. Investor Journey
- **Entry Point:** `/invest-in-aba` (Global Industrial Marketplace)
- **Path:** Home (`/`) -> `/invest-in-aba` -> `/invest-abia` (Global Investment Gateway)
- **CTAs:** "Explore Vision", "Investor Portal", "Register Interest"
- **Conversion Point:** Investment submission form on `/invest-in-aba`
- **Friction Points:** Separate portals (`invest-in-aba` vs `invest-abia`) cause nomenclature confusion.

### 2. Citizen Journey
- **Entry Point:** Homepage (`/`)
- **Path:** Home -> `/serve-abia` (Digital Services) -> `/gada` (Citizen Portal)
- **CTAs:** "Citizen Portal", "Submit Feedback", "Apply for Service"
- **Conversion Point:** Submission of feedback on `/connect-abia`
- **Friction Points:** Multiple routes for digital services (`/serve-abia`, `/serve-abia-2`, `/serve-abia-3`) cause routing duplication.

### 3. Diaspora Journey
- **Entry Point:** `/diaspora-gateway`
- **Path:** Home -> `/diaspora-gateway` -> `/invest-in-aba`
- **CTAs:** "Diaspora Registry", "Explore Investments"
- **Conversion Point:** Diaspora Registry form registration.
