# Navigation Conflict Report

## Identified Navigation Anomalies

1. **Route Suffix Clutter:** Suffix routes (`-2`, `-3`) create a parallel, non-nested route tree which confuses navigation path trackers.
2. **Sidebar-Header Disconnect:** Several dashboards display sidebars that duplicate the top-level header navbar link destinations, causing redundant UI clutter.
3. **Mobile Layout Redirection:** Clicking links on mobile mockups occasionally routes users to desktop pages rather than matching mobile viewport variants.
