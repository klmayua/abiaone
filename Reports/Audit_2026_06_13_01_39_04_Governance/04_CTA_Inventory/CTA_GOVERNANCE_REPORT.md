# CTA Governance Report

## CTA Strategic Issues

1. **Action Overload:** Pages like `/serve-abia` contain **45 CTAs**, causing sensory overload. Recommend a maximum of **3 CTAs** per viewport row.
2. **Styling Inconsistencies:** Custom inline classes style some buttons as gradients, while others use plain green classes. We need a standardized button hierarchy component.
3. **Conversion Tracking:** No telemetry currently measures button clicks. Implement custom hooks in a future phase.
