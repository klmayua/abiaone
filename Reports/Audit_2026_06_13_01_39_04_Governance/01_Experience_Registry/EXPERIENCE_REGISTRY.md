# Experience Registry (Version 2.0)

Defines the core, high-level user experiences across the ABIA ONE platform.

## Experience: discover_abia
- **purpose:** Public tourism, cultural archive, and heritage visibility.
- **target_users:** Citizens, tourists, diaspora, researchers
- **primary_outcome:** Increased interest in local history and tourism engagement.
- **supporting_pages:** `/`
- **supporting_services:** General State Services
- **supporting_dashboards:** None
- **supporting_navigation:** Header Navbar, Floating Directory
- **entry_points:** `/`
- **conversion_points:** `Digital Archives link click`
- **duplicate_experiences:** None
- **governance_notes:** Standardize layout to remove suffix pages.
