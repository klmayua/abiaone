# Experience Hierarchy

Defines the platform organizational blueprint:

```
Experience (e.g., serve_abia)
  └── Category (e.g., Digital Services)
        └── Service (e.g., GADA Portal)
              └── Page (e.g., /gada)
                    └── Action (e.g., Submit Permit Form)
```
## Misplaced Elements Identified
- `/digital-archives-heritage` contains real-time executive server telemetry widgets which belong to `/mda-performance`.
- `/secure-abia` public page contains direct admin security report forms.
