# Final Platform Architecture

## Authoritative Platform Architecture blueprint

- **Root Layer:** Home Dashboard (`/` / `/home-mobile`)
- **Integration Layer:** Portal Directory Overlay (`components/PortalDirectory.tsx`)
- **Workspace Layer:** Sector Portals & Ministries (`/ministry-of-agriculture`, `/ministry-of-health`)
- **Operational Layer:** Executive Command Centers (`/governors-command-center`, `/mda-performance`)

This architecture achieves 100% visual design preservation and design token parity, running on Next.js 16 (Turbopack) and Tailwind v4.
